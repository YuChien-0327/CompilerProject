## Calculater
可支援 pi sin cos tan log ln exp ceil uGuass lGuass floor abs 函式

也可支援連續四則運算和括號優先

可支援正負數定義及轉換

## C_Parser
可支援 string int float double 定義和初始設定

可支援 cin cout 簡單輸入輸出

可支援 型態轉換加(int/float/double/string)減乘除(int/float/double)

可支援 log() sqrt() abs() 函式，型態為 int/float/double

可支援正負數定義轉換及運算

可判斷語意語法正確性
